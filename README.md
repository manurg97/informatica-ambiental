## Introducción

Somos el grupo de 1º de ASIR formado por Manuel, Sergio y Pablo. En este repositorio presentamos un proyecto relacionado con la informática ambiental y el uso de GitHub como herramienta para la creación y publicación de páginas web.

El objetivo principal del proyecto es aprender a crear y estructurar un repositorio en GitHub utilizando archivos HTML, organizar correctamente los contenidos del proyecto y publicar la página mediante GitHub Pages. Además, se incluye información relacionada con la informática ambiental y algunos de sus conceptos principales.

Para realizar este trabajo, cada integrante del grupo ha investigado y desarrollado una parte del contenido. Posteriormente, se ha organizado toda la información en conjunto para darle una estructura clara dentro del repositorio.

## Contenido del proyecto

Cada integrante del grupo ha trabajado en una pregunta específica:

- **Pablo:** ¿Qué es la contaminación ambiental y cuáles son sus campos?  
- **Sergio:** ¿Qué es la informática ecológica?
